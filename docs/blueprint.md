# **App Name**: My Senoner Sarteur

## Core Features:

- Secure Role-Based Authentication: Provides a secure login system with distinct roles (customer, employee, admin) managed through Firebase Authentication and server-side session management.
- Customer Dashboard: Displays a personalized dashboard for customers, including a digital loyalty card (QR code), loyalty point progress, product ordering, wheel of fortune, and profile management.
- Employee Scanner: Offers a minimalistic QR code scanner interface for employees to quickly scan customer loyalty cards, add stamps, and redeem rewards at the point of sale.
- Admin Dashboard and Management: Provides a comprehensive admin dashboard for managing orders, products, customers, and generating seasonal marketing promotions with AI assistance using Google Genkit.
- Product Browsing & Pre-ordering: Allows customers to browse product categories and pre-order items with specified pickup dates, providing a convenient and efficient ordering experience.
- Loyalty Program & Rewards: Enables customers to earn loyalty points, track their progress, and redeem rewards, fostering customer engagement and retention.
- AI-Powered Seasonal Promotion Generator: Integrates Google Genkit to generate marketing promotion ideas based on seasonal themes, available products, and market trends. The system will use these as tools to decide when and how to incorporate information.
- Real-Time Data Updates: Utilizes Firestore Realtime Listeners to provide live updates on order status, loyalty points, and other key data, ensuring a dynamic and engaging user experience.
- Profile Management: Enables customers to manage their profiles, update personal information, and change passwords securely.
- Secure Data Handling: Implements robust security measures, including input validation, authorization checks, and Firestore security rules, to protect user data and prevent unauthorized access.

## Style Guidelines:

- Clean and well-spaced layout with high readability and intuitive navigation. Important functions should always be accessible with as little friction as possible.
- Body font: 'PT Sans' (sans-serif) for clear, accessible information displays. Headline font: 'Playfair' (serif), for elegance in headers.
- Simple, clean icons to represent different sections and actions within the app. Use icons in the brand color.
- Primary color: Deep green (#1B5E20) to convey freshness and local sourcing. Secondary color: Light beige (#F5F5DC) for a clean and natural feel.
- Accent color: Gold (#FFD700) to highlight important actions and call-to-actions.
- Mobile-first design with a responsive layout that adapts seamlessly to different screen sizes and devices.
- Subtle animations and transitions to provide feedback on user interactions and enhance the overall user experience.
